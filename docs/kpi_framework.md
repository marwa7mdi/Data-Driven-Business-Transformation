# Strategic KPI & Measurement Framework

## 🎯 Overview
This document defines the metrics and Key Performance Indicators (KPIs) established to transition the organization into a data-driven culture. The framework was designed to align departmental outputs with the company’s "North Star" goal: **Sustainable Scalability and Operational Excellence.**

---

## 🏛️ 1. Marketing Intelligence KPIs
*Goal: Optimize lead quality and maximize Return on Ad Spend (ROAS).*

| Metric | Definition | Data Source |
| :--- | :--- | :--- |
| **Conversion Rate (CR)** | The percentage of website visitors who become qualified leads in HubSpot. | GA4 + HubSpot API |
| **Cost Per Lead (CPL)** | Total ad spend divided by the number of leads generated. | GTM + CRM Data |
| **Email Engagement (CTR)** | Rate of clicks on call-to-action links within automated email sequences. | Mailster API |
| **ROI per Campaign** | Total revenue generated from a specific marketing source vs. its cost. | SQL Attribution Model |

---

## 💰 2. Sales Performance & Integrity KPIs
*Goal: Ensure revenue predictability and eliminate pipeline manipulation.*

| Metric | Definition | Data Source |
| :--- | :--- | :--- |
| **Sales Velocity** | The average time it takes for a deal to move from 'Created' to 'Closed Won'. | HubSpot CRM |
| **Deal Integrity Index** | Analysis of closing patterns to detect "deal pushing" or target gaming. | Custom SQL Logic |
| **Win Rate** | Percentage of deals closed successfully vs. total deals created. | HubSpot CRM |
| **Customer Acquisition Cost (CAC)** | Total sales and marketing cost to acquire one new customer. | Unified Data Source |

---

## ⚙️ 3. Operational Efficiency KPIs
*Goal: Identify bottlenecks and maximize resource output.*

| Metric | Definition | Data Source |
| :--- | :--- | :--- |
| **Production Throughput** | The volume of service delivery units completed per month. | PostgreSQL (Ops DB) |
| **Bottleneck Identification** | Measuring the "Wait Time" between internal process stages. | Internal Workflow Audit |
| **Employee Productivity Lift** | Increase in output per head after data-driven process optimization. | HR + Ops Data |

---

## 📊 4. Customer Loyalty & Retention
*Goal: Increase Customer Lifetime Value (CLV) through segmentation.*

| Metric | Definition | Data Source |
| :--- | :--- | :--- |
| **RFM Score** | Recency, Frequency, and Monetary value of customer interactions. | Python Analysis |
| **Churn Rate** | The rate at which existing customers stop using the service. | CRM Analytics |

---

## 🛠️ Methodology & Survey Integration
To build this framework, I didn't rely solely on automated data. I conducted **Internal Employee Surveys** to:
1. Identify qualitative pain points in the current workflow.
2. Validate automated data findings with "on-the-ground" reality.
3. Establish a feedback loop between the data department and departmental leads.
